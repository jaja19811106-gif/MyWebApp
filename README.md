# MyWebApp

Java Servlet/JSP Webアプリケーションの学習用プロジェクトです。  
Eclipse + Tomcat + Gradle を利用して開発しています。

## 🚀 実行方法
1. Eclipse でプロジェクトをインポート
2. Tomcat サーバーにデプロイ
3. ブラウザで以下にアクセス  



## 📂 プロジェクト構成
- `src/main/java` : サーブレットのソースコード
- `src/main/webapp` : JSPや静的ファイル
- `WEB-INF/web.xml` : デプロイメント記述子
- `build.gradle` : Gradle ビルド設定

## 📝 学習内容
- サーブレットの基本 (`doGet`, `doPost`)
- JSPとの連携
- DAO/JDBCによるDBアクセス
- Git/GitHubによるバージョン管理
